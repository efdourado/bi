# Sales


## Description

This project is a Power BI dashboard that provides insights into sales performance, trends, and key metrics.
It includes various reports designed to help businesses track revenue, customer behavior, and product performance.

## Structure

- **`sales.pbix`** → Power BI report file with interactive visualizations.
- **`salesData.xlsx`** → Excel sheet containing raw sales data.
- **`salesReport.pdf`** → Exported PDF version for easy sharing.
- **`README.md`** → Documentation for this specific project.

## Key Features

1. **Sales Overview**
   - Total revenue, sales trends, and growth analysis.

2. **Product Performance**
   - Top-selling products and revenue breakdown.

3. **Customer Insights**
   - Customer segmentation and purchasing behavior.
